# Release v0.7.0 - 2026-04-02

## 🎯 主要变更：简化 JWT 认证方案

### 新增功能
- **简化 JWT 认证方案** - 移除了复杂的 NextAuth 依赖，改用简洁的自定义 JWT 认证
- 新增 `src/lib/auth-simple.ts` - 简化版认证模块
- 更新 `src/middleware/auth.ts` - 统一的认证中间件
- 修复了认证流程中的多个问题，提升了性能和稳定性

### 技术改进
- 移除了不必要的依赖，减小了包体积
- 简化了认证流程，更容易维护和调试
- 保持了向后兼容性，现有用户数据不受影响
- 改进了类型安全，减少了 any 类型使用

### 数据模型重构
- 将 `inventory` 表重构为 `inventoryItem` 和 `inventoryLog`
- 移除了不再使用的 `warehouseId` 字段（已移至 `InventoryItem`）
- 清理了旧的迁移文件

### UI/UX 改进
- 新增了报表订阅功能
- 支持导出报表计划任务
- 改进了仪表板布局
- 新增了侧边栏导航组件

### 修复问题
- 修复了认证状态同步问题
- 修复了数据库模型引用错误
- 修复了多个组件的状态管理问题
- 修复了 TypeScript 类型错误
