## 🔧 技术实现

### API 调用

| 接口 | 方法 | 用途 |
|------|------|------|
| `/api/product-research/products` | GET | 获取产品列表（选择器） |
| `/api/product-research/products/[id]` | GET | 获取产品详情 |
| `/api/product-research/comparisons` | POST | 创建对比报告 |
| `/api/product-research/comparisons` | GET | 获取对比历史 |

### 组件依赖

**shadcn/ui 组件：**
- `Button` - 按钮
- `Table` - 对比表格
- `Dialog` - 添加产品对话框
- `Badge` - 状态徽章
- `Card` - 卡片容器
- `Select` - 产品选择器
- `Checkbox` - 多选

**自定义组件：**
- `ProductSelector` - 产品选择器（带搜索）
- `ComparisonTable` - 对比表格
- `DiffHighlight` - 差异高亮组件

### 数据类型

```typescript
interface ComparisonProduct {
  id: string;
  name: string;
  brand: string | null;
  price: number | null;
  category: string;
  attributes: Record<string, string>;
}

interface ComparisonReport {
  id: string;
  productIds: string[];
  products: ComparisonProduct[];
  createdAt: string;
  createdBy: string;
  notes?: string;
}
```
