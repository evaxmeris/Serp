## 🔧 技术实现

### API 调用

| 接口 | 方法 | 用途 |
|------|------|------|
| `/api/product-research/products/[id]` | GET | 获取产品详情（含属性值） |
| `/api/product-research/products/[id]` | PUT | 更新产品信息 |
| `/api/product-research/products/[id]` | DELETE | 删除产品 |
| `/api/product-research/templates?categoryId=xxx` | GET | 获取品类属性模板 |
| `/api/product-research/products/[id]/attributes` | PUT | 批量更新属性值 |

### 组件依赖

**shadcn/ui 组件：**
- `Button` - 按钮
- `Input` - 输入框
- `Label` - 标签
- `Card` - 卡片容器
- `Badge` - 状态徽章
- `Dialog` - 确认对话框
- `Textarea` - 多行文本
- `Select` - 下拉选择
- `Separator` - 分隔线
- `Tabs` - 标签页（可选）

**自定义组件：**
- `AttributeDisplay` - 动态属性展示组件
