## 🔧 技术实现

### 依赖库

```bash
npm install recharts
```

### 组件依赖

**recharts 组件：**
- `BarChart` - 柱状图
- `PieChart` - 饼图
- `LineChart` - 折线图
- `ResponsiveContainer` - 响应式容器

**shadcn/ui 组件：**
- `Card` - 卡片容器
- `Progress` - 进度条
- `Badge` - 徽章
- `Select` - 筛选器

### API 调用

| 接口 | 方法 | 用途 |
|------|------|------|
| `/api/product-research/products` | GET | 获取产品数据（统计用） |
| `/api/product-research/stats/overview` | GET | 获取概览统计数据 |
| `/api/product-research/stats/category` | GET | 获取品类分布 |
| `/api/product-research/stats/price` | GET | 获取价格分布 |
| `/api/product-research/stats/margin` | GET | 获取毛利率统计 |
