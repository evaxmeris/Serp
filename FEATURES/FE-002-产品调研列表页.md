## 🔧 技术实现

### API 调用

| 接口 | 方法 | 用途 |
|------|------|------|
| `/api/product-research/products` | GET | 获取产品列表（分页 + 筛选） |
| `/api/product-research/products` | POST | 创建新产品 |
| `/api/product-research/products/[id]` | GET | 获取产品详情 |
| `/api/product-research/products/[id]` | PUT | 更新产品 |
| `/api/product-research/products/[id]` | DELETE | 删除产品 |
| `/api/product-research/categories` | GET | 获取品类列表（树形） |

### 组件依赖

**shadcn/ui 组件：**
- `Button` - 按钮
- `Input` - 搜索框

- `Select` - 品类筛选下拉

### 数据类型

```typescript
interface ProductResearch {
  id: string;
  name: string;           // 产品名称
  code: string;           // 产品编码
  categoryId: string;     // 品类 ID
  category?: ProductCategory; // 关联品类
  brand?: string;         // 品牌
  price?: number;         // 价格
  currency?: string;      // 货币单位（USD/CNY）
  imageUrl?: string;      // 产品图片
  sourceUrl?: string;     // 来源链接
 调研日期：Date;
  status: 'draft' | 'in_progress' | 'completed';
  notes?: string;         // 备注
  attributes?: ProductAttributeValue[]; // 动态属性值
  createdAt: Date;
  updatedAt: Date;
}

interface ListFilters {
  categoryId?: string;    // 品类筛选
  search?: string;        // 搜索关键词
  status?: string;        // 状态筛选
  page?: number;          // 页码
  pageSize?: number;      // 每页数量
  sortBy?: string;        // 排序字段
  sortOrder?: 'asc' | 'desc'; // 排序方向
}
```
