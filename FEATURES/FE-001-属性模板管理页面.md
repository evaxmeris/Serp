### API 调用

| 接口 | 方法 | 用途 |
|------|------|------|
| `/api/product-research/categories` | GET | 获取品类列表（树形结构） |
| `/api/product-research/templates` | GET | 获取属性模板列表（按品类过滤） |
| `/api/product-research/templates` | POST | 创建属性模板 |
| `/api/product-research/templates/[id]` | PUT | 更新属性模板 |
| `/api/product-research/templates/[id]` | DELETE | 删除属性模板 |

- `draggable` 属性启用拖拽
- `onDragStart` / `onDragEnd` / `onDragOver` / `onDrop` 事件处理
- 拖拽后调用 API 更新 `sortOrder` 字段
