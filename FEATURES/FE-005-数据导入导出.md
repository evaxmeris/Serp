## 🔧 技术实现

### API 调用

| 接口 | 方法 | 用途 |
|------|------|------|
| `/api/product-research/import/template` | GET | 下载导入模板 |
| `/api/product-research/import/preview` | POST | 预览导入数据 |
| `/api/product-research/import` | POST | 执行批量导入 |
| `/api/product-research/export` | GET | 导出数据（CSV/Excel） |

### 依赖库

```json
{
  "xlsx": "^0.18.5",
  "papaparse": "^5.4.1"
}
```

### 数据类型

```typescript
interface ImportPreview {
  totalRows: number;
  validRows: number;
  invalidRows: number;
  data: ImportRow[];
}

interface ImportRow {
  rowNumber: number;
  data: Record<string, string>;
  isValid: boolean;
  errors: string[];
}

interface ExportOptions {
  format: 'csv' | 'excel';
  categoryIds?: string[];
  status?: string[];
  productIds?: string[];
}
```
