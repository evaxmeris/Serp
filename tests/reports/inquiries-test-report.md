## 🔧 发现的问题与修复

### 问题 1: 缺少 [id] 路由文件
- **发现**: `src/app/api/inquiries/[id]/route.ts` 不存在
- **影响**: 无法获取/更新/删除单个询盘
- **修复**: 创建完整的 [id] 路由，包含 GET/PUT/DELETE 方法
- **状态**: ✅ 已修复

### 问题 2: Prisma 模型关系不匹配
- **发现**: FollowUp 模型没有 `user` 关系，Inquiry 模型没有 `quotations` 关系
- **影响**: API 查询失败，返回 500 错误
- **修复**: 移除不存在的 include 字段
- **状态**: ✅ 已修复

### 问题 3: 状态枚举值不匹配
- **发现**: 测试使用了不存在状态值 `IN_PROGRESS` 和 `CLOSED`
- **实际枚举**: NEW, CONTACTED, QUOTED, NEGOTIATING, WON, LOST
- **修复**: 更新测试用例使用正确的状态值
- **状态**: ✅ 已修复
