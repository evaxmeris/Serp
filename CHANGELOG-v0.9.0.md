### 1. 认证系统修复（P0）

**问题回顾：**

- 导致 API 直接访问返回 401 错误

**修复内容：**
- ✅ 修改 `auth-simple.ts` 动态获取 JWT 密钥，解决模块缓存问题
- ✅ 修改 `auth-api.ts` 使用相同的动态获取方式
- ✅ 更新所有模块 API 路由统一使用 `auth-simple.ts` 认证

### 2. 批量操作扩展（P1）

**供应商管理批量操作（新增）：**
- ✅ 批量导入（CSV/Excel）- 单次最多 500 条
- ✅ 批量导出（CSV）- 支持筛选
- ✅ 批量分类调整 - 快速修改供应商分类

**库存管理批量操作（新增）：**
- ✅ 批量入库 - 支持多个商品同时入库
- ✅ 批量出库 - 支持多个商品同时出库
- ✅ 批量盘点 - 快速更新库存数量

**新增 API 端点（共 6 个）：**
```
POST /api/suppliers/batch-import
GET /api/suppliers/batch-export
POST /api/suppliers/batch-category
POST /api/inventory/batch-receive
POST /api/inventory/batch-issue
POST /api/inventory/batch-count
```

**状态：** ✅ 100% 完成
