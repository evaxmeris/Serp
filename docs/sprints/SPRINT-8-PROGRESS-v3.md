## 🧮 利润核算方案（最终版）

**利润计算公式（按订单）：**
```
订单利润 = 销售收入
         - 采购成本
         - 物流运费
         - 平台佣金
         - 广告费用
         - 退款金额
         - 其他费用
```

**利润维度：**
- 按订单明细核算
- 按 SKU 汇总统计
- 按平台汇总统计
- 按业务员汇总统计
- 按时间段汇总统计

**成本分摊规则：**
- 固定成本：采购成本直接分摊到 SKU
- 变动成本：运费按重量/金额分摊
- 平台费用：按实际佣金比例扣除
- 广告费用：按订单归属分摊

## 💰 绩效提成方案（最终版）

**提成计算方式：**
1. **个人业绩提成** = 个人订单利润 × 提成比例
2. **阶梯式提成比例**：
   - 月利润 ≤ 1万：提成比例 5%
   - 1万 < 月利润 ≤ 3万：提成比例 8%
   - 3万 < 月利润 ≤ 5万：提成比例 10%
   - 月利润 > 5万：提成比例 12%

3. **团队业绩提成** = 团队总利润 × 团队提成比例（由团队经理分配）
4. **提成结算周期**：按月结算，客户确认收货后计入

**权限：**
- 运营专员只能看自己的业绩和提成
- 绩效专员可以看所有人业绩，负责核算
- 创始人/财务可以看全部数据
