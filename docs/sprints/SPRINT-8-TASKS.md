## ✅ 验收标准

### 功能验收

1. **权限管理**
   - [ ] 支持 5 种预定义角色（创始人/运营/业务员/美工/采购）
   - [ ] 支持自定义角色创建和权限分配
   - [ ] 业务员只能查看自己的客户和订单
   - [ ] 操作日志完整记录

2. **阿里国际站订单同步**
   - [ ] 支持账号 OAuth 授权
   - [ ] 每 15 分钟自动同步新订单
   - [ ] 订单状态双向同步
   - [ ] 支持手动触发同步

3. **多仓库管理**
   - [ ] 可创建多个仓库
   - [ ] 仓库类型区分（本地/海外/第三方）
   - [ ] 库存按仓库隔离
   - [ ] 支持仓库间库存查询

4. **产品刊登**
   - [ ] 产品可关联多平台
   - [ ] 一键刊登到目标平台
   - [ ] 刊登状态实时跟踪

### 技术验收

- [ ] 数据库向后兼容，不破坏现有数据
- [ ] 所有 API 有完整的错误处理
- [ ] 单元测试覆盖率 ≥ 80%
- [ ] 核心流程集成测试通过
- [ ] 代码注释完整符合规范
- [ ] 文档更新完成
