## 📦 发布内容

### 补丁修复：客户管理弹窗 Label 缺失

**问题描述**:
- 客户管理页面的「新增客户」和「编辑客户」弹窗中，输入框仅有 placeholder 提示，没有 Label 标签
- 在手机端输入时，placeholder 被键盘遮挡，用户无法知道自己正在填写哪个字段
- 影响可用性，特别是移动端用户体验

**影响范围**:
- `/customers` 页面 → 新增弹窗
- `/customers` 页面 → 编辑弹窗
- 5 个输入字段都缺失标签：公司名称、联系人、邮箱、电话、国家/地区

**修复方案**:
1. 添加 `Label` 组件导入
2. 为每个输入框添加包装 `<div className="space-y-2"><Label ...><Input ...></div>`
3. 使用 `htmlFor` 关联 Label 和 Input，提升可访问性
4. 修改 placeholder 文案使其更友好（原来 placeholder 就是标签，现在分开后placeholder改为提示性文字）

**修改文件**:
- `src/app/customers/page.tsx` → 添加 Label 导入，为新增/编辑弹窗所有输入框添加 Label

**代码变更**:
- 新增：1 个 import
- 新增：10 个 Label 组件（5个新增 + 5个编辑）
- 修改：placeholder 文案
- 无删除，无破坏性变更
