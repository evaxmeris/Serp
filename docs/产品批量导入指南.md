# Trade ERP 产品批量导入指南

## 一、快速开始

### 1.1 操作步骤

1. 登录 Trade ERP 系统，进入 **产品管理** 页面
2. 点击页面上方的 **批量导入** 按钮
3. 下载导入模板（点击"下载导入模板"按钮）
4. 按照模板格式填写产品数据
5. 将填写好的文件拖拽或上传到导入界面
6. 预览数据，勾选需要导入的记录
7. 选择 SKU 已存在时的处理方式（更新 / 跳过）
8. 点击确认导入

### 1.2 支持的文件格式

| 格式 | 支持情况 | 说明 |
|------|----------|------|
| CSV | ✅ 支持 | 推荐使用，编码要求 UTF-8 |
| Excel (.xlsx) | ⚠️ 前端显示支持 | 实际解析为 CSV 方式 |

> 建议：在 Excel 中编辑后，**另存为 CSV (UTF-8 编码)** 再导入。

## 二、CSV 表格列要求

### 2.1 列定义

表格第一行为表头（列名），系统会根据表头关键词**模糊匹配**对应字段。以下是支持的列：

| 列名（表头关键词） | 必填 | 数据类型 | 说明 | 示例值 |
|---|---|---|---|---|
| SKU | **是** | 文本 | 产品唯一标识，不能重复 | `SKU001`、`PRO-2024-001` |
| 产品名称 / name | **是** | 文本 | 产品中文名称 | `不锈钢保温杯` |
| 分类 / category | 否 | 文本 | 产品分类名称 | `日用品`、`电子产品` |
| 成本价格 / cost price | 否 | 数字 | 产品成本价（正数或0） | `10.50`、`0` |
| 销售价格 / sale price | 否 | 数字 | 产品销售价（正数或0） | `20.00` |
| 最小起订量 / min order | 否 | 整数 | MOQ，最小订货数量 | `100` |
| 供应商 / supplier | 否 | 文本 | 供应商名称 | `深圳XX贸易公司` |
| 重量 / weight | 否 | 数字 | 产品重量（单位：kg） | `0.5` |
| 长度 / length | 否 | 数字 | 产品长度（单位：cm） | `10` |
| 宽度 / width | 否 | 数字 | 产品宽度（单位：cm） | `10` |
| 高度 / height | 否 | 数字 | 产品高度（单位：cm） | `5` |

### 2.2 表头关键词匹配规则

系统对表头进行**模糊匹配**，不区分中英文。以下写法都可以：

- `SKU` → 匹配 SKU
- `产品名称` 或 `Name` → 匹配产品名称
- `分类` 或 `Category` → 匹配分类
- `成本价格` 或 `Cost Price` 或 `成本价` → 匹配成本价格
- `销售价格` 或 `Sale Price` 或 `销售价` → 匹配销售价格
- `最小起订量` 或 `Min Order` 或 `起订量` → 匹配最小起订量
- `供应商` 或 `Supplier` → 匹配供应商
- `重量` 或 `Weight` → 匹配重量
- `长度` 或 `Length` → 匹配长度
- `宽度` 或 `Width` → 匹配宽度
- `高度` 或 `Height` → 匹配高度

## 三、CSV 模板示例

### 3.1 标准模板（推荐直接复制使用）

```csv
SKU,产品名称,分类,成本价格,销售价格,最小起订量,供应商,重量,长度,宽度,高度
SKU001,不锈钢保温杯,日用品,10.00,20.00,100,深圳XX贸易公司,0.5,10,10,5
SKU002,蓝牙耳机,电子产品,15.00,30.00,50,东莞XX电子厂,1.0,15,15,8
SKU003,瑜伽垫,运动用品,8.00,15.00,200,义乌XX体育用品,0.8,180,60,1
```

### 3.2 最小必填模板

```csv
SKU,产品名称
PRO-A001,白色T恤
PRO-A002,黑色裤子
```

```csv

SKU001,保温杯,日用品,10,20,100,供应商A,0.5,10,10,5
```

## 四、字段详细说明

### 4.1 SKU（必填）

- **格式**：自由文本，最大 100 字符
- **唯一性**：系统中 SKU 必须唯一
- **命名建议**：
  - 使用字母+数字组合，如 `SKU001`、`PRO-2024-001`
  - 避免使用中文、特殊字符
  - 建议使用有意义的编码规则，如：`类目-序号`（`DM-001` 表示日用品第1号）

### 4.2 产品名称（必填）

- **格式**：文本，最大 200 字符
- **说明**：产品的中文名称

### 4.3 价格字段

| 字段 | 说明 | 格式 |
|------|------|------|
| 成本价格 | 产品的采购/生产成本 | 数字，保留2位小数，如 `10.50` |
| 销售价格 | 产品的销售价格 | 数字，保留2位小数，如 `20.00` |

> 注意：价格必须 ≥ 0，不能为负数。如果留空，默认为 0。

### 4.4 分类

- 填写产品所属的分类名称
- 如果分类不存在，导入后该产品的分类将为空
- 建议先在系统中创建好产品分类，再进行导入

### 4.5 供应商

- 填写供应商名称
- 此字段仅用于记录，不会自动创建供应商记录

### 4.6 重量和尺寸

| 字段 | 单位 | 格式 |
|------|------|------|
| 重量 | kg（千克） | 数字，如 `0.5` |
| 长度 | cm（厘米） | 数字，如 `10` |
| 宽度 | cm（厘米） | 数字，如 `10` |
| 高度 | cm（厘米） | 数字，如 `5` |

## 五、导入模式

### 5.1 新建模式（默认）

- SKU 不存在时：创建新产品
- SKU 已存在时：根据选择**更新**或**跳过**

### 5.2 SKU 已存在时的处理选项

| 选项 | 行为 |
|------|------|
| **更新** | 用导入的数据覆盖现有产品信息 |
| **跳过** | 保留现有数据，不覆盖 |

## 六、注意事项和常见问题

### 6.1 CSV 编码

- 必须使用 **UTF-8 编码**
- Windows Excel 默认保存为 GBK 编码，会导致中文乱码
- **正确做法**：在 Excel 中点击"文件 → 另存为 → CSV UTF-8 (逗号分隔)"

### 6.2 逗号分隔符

- CSV 文件中字段之间用**英文逗号** `,` 分隔
- 如果单元格内容包含逗号，需要用双引号包裹，如 `"红色,蓝色"`

### 6.3 数字格式

- 不要使用千分位分隔符：`1000` ✅，`1,000` ❌
- 小数使用英文点号：`10.50` ✅，`10,50` ❌

### 6.4 空值处理

- 选填字段可以留空（单元格为空即可）
- 必填字段（SKU、产品名称）为空时，该行会被标记为错误，不会被导入

### 6.5 文件大小限制

- 最大支持 **10MB** 的 CSV 文件
- 单次导入建议不超过 **1000 条**记录

### 6.6 导入后验证

- 导入完成后会显示结果统计：成功数、失败数、跳过数
- 如果有失败记录，可以下载错误报告查看具体原因
- 导入成功后到"产品管理"页面确认数据是否正确

## 七、常见问题 FAQ

### Q: 导入后产品名称显示乱码？
A: 这是因为 CSV 文件不是 UTF-8 编码。请用 Excel 重新另存为 "CSV UTF-8" 格式。

### Q: 导入提示 "SKU 已存在"？
A: 说明该 SKU 在系统中已有记录。可以选择"更新"模式覆盖，或修改 CSV 中的 SKU。

### Q: 可以只导入部分列吗？
A: 可以。至少包含 SKU 和 产品名称 两列即可，其他列可选。

### Q: 导入失败后数据会丢失吗？
A: 不会。导入是逐条处理的，成功的记录会保留，失败的不会影响其他记录。

### Q: 能否通过导出现有产品作为模板？
A: 可以。在产品管理页面选择产品后，点击"批量导出"，下载 CSV 文件后修改再导入。
