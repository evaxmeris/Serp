## 快速开始

### 访问地址

- 开发环境: `http://localhost:3001`
- 生产环境: 根据实际部署地址

### 进入登录页

在浏览器地址栏输入系统地址，会自动跳转到登录页。

登录页地址: `/login`

## 登录系统

### 步骤 1: 打开登录页

你会看到如下登录界面：

```
┌─────────────────────────────────────┐
│         Trade ERP                   │
│       外贸业务管理专家              │
│                                     │
│  [邮箱         ]                    │
│  [密码         ]                    │
│                                     │
│  [        登录        ]             │
│                                     │
│  没有账户？立即注册 →               │
└─────────────────────────────────────┘
```

### 步骤 2: 输入登录信息

- **邮箱**: 输入你注册的邮箱地址
- **密码**: 输入你的账户密码

### 步骤 3: 点击登录

点击「登录」按钮，系统会验证你的信息。

### 登录成功

如果邮箱和密码正确，系统会自动跳转到 **仪表板** (`/dashboard`)，你可以开始使用系统了。

### 登录失败

登录失败会显示错误信息：

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 账号或密码错误 | 邮箱不存在或密码不正确 | 请检查输入，重新尝试 |
| 账户等待审批 | 新注册账户还未被管理员批准 | 请联系管理员审批 |
| 账户已暂停 | 你的账户被暂时停用 | 请联系管理员解除暂停 |
| 账户已禁用 | 你的账户被禁用 | 请联系管理员了解原因 |
| 请求过于频繁，请稍后再试 | 登录失败次数太多，触发了限流 | 请等待 15 分钟后再试 |

## 登出系统

### 退出登录步骤

1. 在仪表板页面，点击右上角用户头像/用户名
2. 在下拉菜单中选择「退出登录」
3. 系统会清除你的登录信息，返回登录页

### 为什么要登出？

- 在公共电脑上使用后，一定要登出，防止他人访问你的账户
- 如果你换账户登录，需要先退出当前账户

## 用户角色与权限

系统支持四种用户角色，不同角色有不同的权限：

### 角色说明

| 角色 | 中文 | 权限范围 | 适合人群 |
|------|------|----------|----------|
| `VIEWER` | 访客 | 只读权限，只能查看不能修改 | 客户、访客 |
| `USER` | 普通用户 | 可以操作自己负责的业务 | 业务员、采购员 |
| `MANAGER` | 经理 | 可以管理部门业务，审批订单 | 部门经理 |
| `ADMIN` | 管理员 | 完整权限，包括系统配置、用户管理 | 系统管理员 |

### 权限等级

权限从低到高：
```
VIEWER < USER < MANAGER < ADMIN
```

高级角色自动拥有低级角色的所有权限。

### 如何查看我的角色？

登录后，在仪表板右上角可以看到你的用户名和角色信息。

## 常见问题

### Q: 忘记密码怎么办？

**A**: 当前版本还未实现自助密码重置功能。请联系系统管理员重置密码。

### Q: 登录后显示"未认证"怎么办？

**A**: 可能的原因：
- Cookie 被浏览器清除了
- 会话已过期（默认 7 天）

**解决方法**: 重新登录即可。

### Q: 为什么我不能访问某些页面？

**A**: 这是权限控制。你的角色权限不足，无法访问该页面。请联系管理员升级你的权限。

### Q: 如何注册新账户？

**A**: 
1. 在登录页点击「没有账户？立即注册 →」
2. 填写注册信息
3. 提交后等待管理员审批
4. 审批通过后即可登录

> 注意：有些部署可能关闭了开放注册，需要联系管理员创建账户。

### Q: 登录会保持多久？

**A**: 默认保持登录 7 天。7 天后需要重新登录。如果你主动登出，会话立即失效。

### Q: 可以在多台电脑同时登录吗？

**A**: 可以。当前没有限制同时登录的设备数量。

### Q: 如何修改密码？

**A**: 当前版本还未实现修改密码功能。请联系系统管理员修改。

## 安全建议

1. **使用强密码**
   - 密码长度至少 8 位
   - 包含大小写字母和数字
   - 不要使用常见单词
   - 不要使用与你其他网站相同的密码

2. **不要共享账户**
   - 每个人使用自己的账户
   - 方便审计和权限管理

3. **公共设备使用后务必登出**
   - 如果在公司共用电脑、网吧等地方使用
   - 使用完一定要点击退出登录

4. **保护好你的邮箱**
   - 邮箱是找回密码的重要途径
   - 确保你的邮箱密码安全
