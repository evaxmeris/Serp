## 📋 使用说明

本检查清单用于订单管理和采购管理模块的手动测试。测试前请确保：

- [ ] 测试环境已启动 (`npm run dev`)
- [ ] 测试数据库已连接
- [ ] 测试数据已准备 (`npm run test:data:prepare`)
- [ ] 浏览器已打开 http://localhost:3000
