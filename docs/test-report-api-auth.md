## Bug 清单

### 已修复的 Bug
1. **BUG-AUTH-001**: JWT 密钥不一致导致认证失败
   - **根因**: `auth-simple.ts` 和 `auth-api.ts` 模块使用静态常量存储密钥，模块缓存导致密钥不一致
   - **解决方案**: 改为动态获取密钥，每次使用时从 `process.env.NEXTAUTH_SECRET` 读取
   - **状态**: ✅ 已修复

### 待优化的问题
1. **PERF-001**: 单个资源查询接口响应时间过长 (~480ms)
   - **影响**: `/api/customers/1`, `/api/v1/inbound-orders/1` 等接口
   - **建议**: 添加数据库索引优化查询性能

2. **API-001**: 部分批量操作接口返回 405 (Method Not Allowed)
   - **影响**: `/api/customers/batch-import`, `/api/customers/batch-tag` 等
   - **原因**: 这些接口可能只支持 POST 方法，测试使用了 GET
   - **建议**: 更新测试脚本使用正确的 HTTP 方法

3. **API-002**: 报表接口需要查询参数
   - **影响**: `/api/v1/reports/sales`, `/api/v1/reports/profit` 等返回 400
   - **原因**: 这些接口需要日期范围等参数
   - **建议**: 在测试中提供必要的查询参数
