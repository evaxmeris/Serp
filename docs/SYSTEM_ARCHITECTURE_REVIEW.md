## 二、外贸ERP标准功能架构

参照成熟外贸ERP系统，完整的业务流程应覆盖：

```
供应商开发 → 产品选品 → 采购下单 → 收货质检 → 入库上架
                                              ↓
客户开发 → 报价 → 订单 → 出库拣货 → 物流发货 → 财务结算
                                              ↓
                    报表分析 ← 数据仪表盘 ← 平台同步
```

### 标准一级模块

```
1. 数据仪表盘       - 核心 KPI 一览
2. 客户管理         - 客户信息 / 联系人 / 跟进记录
3. 供应商管理       - 供应商 / 评估 / 付款记录
4. 产品管理         - 产品列表 / 品类 / 属性模板
5. 产品开发         - 市场调研 / 竞品分析 / 产品对比 / 数据导入
6. 报价管理         - 报价单 / 转订单
7. 订单管理         - 销售订单 / 平台订单
8. 采购管理         - 采购订单 / 收货确认 / 采购入库
9. 仓储物流         - 库存 / 出库 / 物流服务商 / 物流订单 / 发货
10. 财务管理         - 应收应付 / 收付款 / 报销 / 成本核算
11. 报表中心         - 利润 / 销售 / 库存 / 现金流 / 采购
12. 系统设置         - 用户权限 / 审批流程 / 仓库 / 平台对接 / 数据同步
```

## 三、优化方案

### 3.1 导航结构重组

```
📊 仪表盘
  └─ 数据仪表盘

👥 客户管理
  └─ 客户列表（含创建/编辑/删除/导入/导出）
  └─ 客户跟进（询盘记录）

🏭 供应商管理
  └─ 供应商列表
  └─ 供应商评估

📦 产品管理
  └─ 产品列表
  └─ 品类管理
  └─ 属性模板

🔬 产品开发
  └─ 调研看板
  └─ 产品调研
  └─ 竞品分析        ← 🆕 新增页面
  └─ 产品对比
  └─ 数据导入

💰 报价管理 ─────── 独立为一级模块
  └─ 报价列表
  └─ 新建报价

📋 订单管理
  └─ 订单列表
  └─ 新建订单

🛒 采购管理
  └─ 采购订单
  └─ 收货确认        ← 🆕 新增页面（或合并到采购订单详情）
  └─ 采购入库

🚚 仓储物流
  └─ 库存管理
  └─ 出库管理
  └─ 物流服务商
  └─ 物流订单
  └─ 发货记录

💵 财务管理          ← 🆕 全新模块
  └─ 应收应付
  └─ 收付款记录
  └─ 费用报销
  └─ 利润核算

📈 报表中心
  └─ 利润报表
  └─ 销售报表
  └─ 库存报表
  └─ 采购报表
  └─ 现金流报表
  └─ 订阅管理

⚙️ 系统设置
  └─ 用户及权限
  └─ 角色管理
  └─ 审批流程
  └─ 仓库配置
  └─ 平台账号        ← 🆕 新增页面
  └─ 数据同步
```
