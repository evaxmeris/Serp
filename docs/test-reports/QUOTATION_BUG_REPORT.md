## 🐛 发现的 Bug

### BUG-QUOTATION-002: 报价单详情页类型错误

**优先级：** P0  
**状态：** 🔴 待修复  
**发现时间：** 2026-03-09 01:40

**现象：**
- 访问报价单详情页显示 Runtime TypeError
- 错误信息：`quotation.totalAmount.toFixed is not a function`
- 页面无法显示，只显示错误覆盖层

**复现步骤：**
1. 访问 http://localhost:3001/quotations
2. 点击任意报价单的操作按钮（查看）
3. 页面显示错误

**错误位置：**
```
文件：src/app/quotations/[id]/page.tsx
行号：300
代码：{quotation.currency} {quotation.totalAmount.toFixed(2)}
```

**根本原因：**
- API 返回的 `totalAmount` 是字符串类型（如 `"2050"`）
- 前端代码直接调用 `.toFixed(2)` 方法
- 字符串没有 `.toFixed()` 方法，导致运行时错误

**影响范围：**
- `/quotations/[id]` - 所有报价单详情页

**修复方案：**

1. **修改接口定义：**
```typescript
interface Quotation {
  // ...
  totalAmount: string | number;
  // ...
}
```

2. **添加格式化函数：**
```typescript
const formatAmount = (amount: string | number) => {
  const num = typeof amount === 'string' ? parseFloat(amount) : amount;
  return num.toFixed(2);
};

// 使用
{quotation.currency} {formatAmount(quotation.totalAmount)}
```

3. **检查所有使用 totalAmount 的地方：**
- 详情页
- 编辑页
- 列表页（可能已有修复）

**预计修复时间：** 15 分钟
