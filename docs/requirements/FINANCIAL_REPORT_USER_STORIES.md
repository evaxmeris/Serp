## US-FR-001 利润报表

**角色：** 财务经理/运营总监  
**功能：** 查看按产品/品类/时间段的利润报表  
**价值：** 分析哪些产品最赚钱，优化产品策略  

**验收标准：**
- [ ] 支持按产品维度查看利润
- [ ] 支持按品类维度查看利润
- [ ] 支持按时间段筛选（日/周/月/季/年）
- [ ] 显示毛利润、毛利率、净利润
- [ ] 支持数据导出 Excel

**优先级：** P0  
**依赖模块：** 产品管理、销售管理、采购管理、库存管理

**用户故事描述：**

作为财务经理，我希望能够查看不同维度的利润报表，以便了解哪些产品和品类最赚钱，从而优化产品组合和定价策略。

**场景示例：**
1. 查看 TikTok 渠道上周各产品的利润情况，发现某款夜光笔毛利率高达 60%，决定加大推广力度
2. 对比各品类的季度利润，发现家居品类利润率下降，需要分析原因并调整采购策略
3. 导出年度利润报表，用于年度经营总结和下一年度预算规划

## US-FR-002 销售分析报表

**角色：** 销售主管/运营专员  
**功能：** 查看销售数据的各项指标和趋势分析  
**价值：** 监控销售业绩，发现销售机会和问题  

**验收标准：**
- [ ] 显示销售额趋势图（日/周/月）
- [ ] 显示销售量排名（Top 50 产品）
- [ ] 显示客单价和转化率
- [ ] 显示退货率分析
- [ ] 支持按渠道对比销售数据
- [ ] 支持数据导出

**优先级：** P0  
**依赖模块：** 销售管理、产品管理

**用户故事描述：**

作为销售主管，我希望能够实时查看销售数据的各项指标，以便及时发现销售趋势变化，调整运营策略。

**场景示例：**
1. 每天早上查看昨日销售数据，了解整体销售情况
2. 发现某产品销量突然下降，及时排查原因（差评？断货？竞争？）
3. 对比各渠道销售表现，优化渠道资源分配

## US-FR-003 采购成本分析报表

**角色：** 采购经理/财务经理  
**功能：** 分析采购成本结构和变化趋势  
**价值：** 控制采购成本，优化供应商管理  

**验收标准：**
- [ ] 显示采购成本按供应商汇总
- [ ] 显示采购成本按品类汇总
- [ ] 显示采购单价趋势分析
- [ ] 显示供应商价格对比
- [ ] 成本异常时发出预警
- [ ] 支持数据导出

**优先级：** P1  
**依赖模块：** 采购管理、产品管理、财务管理

**用户故事描述：**

作为采购经理，我希望能够分析采购成本的变化趋势，以便与供应商谈判降价，控制采购成本。

**场景示例：**
1. 发现某供应商的采购单价连续 3 个月上涨，启动价格谈判
2. 对比多家供应商的同类产品价格，优化供应商选择
3. 分析各品类的采购成本占比，识别成本控制重点

---

## US-FR-004 库存周转分析报表

**角色：** 库存主管/运营总监  
**功能：** 分析库存周转效率和库龄结构  
**价值：** 优化库存结构，减少资金占用  

**验收标准：**
- [ ] 显示库存周转率（按 SKU/品类）
- [ ] 显示库存天数分析
- [ ] 识别滞销库存（90 天无销售）
- [ ] 预警缺货风险
- [ ] 显示库龄分布
- [ ] 支持数据导出

**优先级：** P1  
**依赖模块：** 库存管理、销售管理、产品管理

**用户故事描述：**

作为库存主管，我希望能够分析库存周转情况，以便及时处理滞销库存，避免缺货风险，提高资金使用效率。

**场景示例：**
1. 发现某 SKU 库存天数超过 180 天，启动促销清仓
2. 监控畅销品的库存水平，避免断货影响销售
3. 分析各品类的库存周转率，优化备货策略

## US-FR-005 现金流报表

**角色：** 财务总监/老板  
**功能：** 追踪现金流入流出情况，预测现金流趋势  
**价值：** 确保公司资金安全，避免资金链断裂  

**验收标准：**
- [ ] 显示经营活动现金流
- [ ] 显示投资活动现金流
- [ ] 显示筹资活动现金流
- [ ] 预测未来 30/60/90 天现金流
- [ ] 显示资金缺口预警
- [ ] 支持数据导出

**优先级：** P0  
**依赖模块：** 财务管理、销售管理、采购管理

**用户故事描述：**

作为财务总监，我希望能够实时监控现金流状况并预测未来趋势，以便提前安排资金，避免资金链断裂风险。

**场景示例：**
1. 发现未来 30 天可能出现资金缺口，提前安排融资
2. 分析经营活动现金流为正，确认业务健康
3. 规划大额采购付款时间，避免影响日常运营

## US-FR-006 应收/应付账款报表

**角色：** 财务经理/会计  
**功能：** 管理应收账款和应付账款  
**价值：** 优化资金周转，降低坏账风险  

**验收标准：**
- [ ] 显示应收账款账龄分析
- [ ] 显示应付账款账龄分析
- [ ] 逾期账款提醒
- [ ] 显示客户信用分析
- [ ] 生成供应商付款计划
- [ ] 支持数据导出

**优先级：** P1  
**依赖模块：** 财务管理、销售管理、采购管理

**用户故事描述：**

作为财务经理，我希望能够管理应收和应付账款，以便及时催收逾期款项，合理安排付款计划，优化资金周转。

**场景示例：**
1. 发现某客户账款逾期 60 天，启动催收流程
2. 根据供应商账期安排付款计划，充分利用信用期
3. 分析客户信用状况，调整赊销政策

## US-FR-007 费用分析报表

**角色：** 财务经理/运营总监  
**功能：** 分析各项费用支出情况  
**价值：** 控制费用支出，提高盈利能力  

**验收标准：**
- [ ] 显示物流费用分析（按渠道/目的地）
- [ ] 显示营销费用分析（按渠道/活动）
- [ ] 显示平台费用分析（佣金/广告/仓储）
- [ ] 显示人力成本分析
- [ ] 费用异常时发出预警
- [ ] 支持数据导出

**优先级：** P1  
**依赖模块：** 财务管理、物流管理、销售管理

**用户故事描述：**

作为财务经理，我希望能够分析各项费用的支出情况，以便发现费用异常，控制成本，提高盈利能力。

**场景示例：**
1. 发现某渠道物流费用占比异常高，排查原因并优化
2. 分析营销活动的 ROI，优化广告投放策略
3. 对比各月费用变化趋势，识别成本控制机会

## US-FR-008 经营仪表盘

**角色：** 老板/运营总监/财务经理  
**功能：** 综合展示核心经营指标  
**价值：** 快速了解公司整体经营状况  

**验收标准：**
- [ ] 显示核心 KPI 卡片（GMV、毛利、净利率等）
- [ ] 显示趋势图表（折线图/柱状图）
- [ ] 显示排名榜单（产品/品类/渠道）
- [ ] 显示预警信息
- [ ] 支持自定义指标配置
- [ ] 支持多端访问（PC/手机）

**优先级：** P0  
**依赖模块：** 所有上游模块

**用户故事描述：**

作为老板，我希望能够在一个页面上看到公司所有核心经营指标，以便快速了解公司整体经营状况，及时发现需要关注的问题。

**场景示例：**
1. 每天早上打开手机查看经营仪表盘，了解昨日关键指标
2. 发现净利率指标变红（低于目标），深入查看利润报表分析原因
3. 在周会上展示经营仪表盘，与团队讨论经营情况
