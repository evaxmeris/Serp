## 详细验收标准

### AC-001: 品类选择功能

**对应故事：** US-001

#### 测试场景

| 场景编号 | 场景描述 | 预期结果 |
|----------|----------|----------|
| TS-001-01 | 打开创建调研页面 | 显示品类下拉选择框 |
| TS-001-02 | 点击品类下拉框 | 显示所有启用的品类列表 |
| TS-001-03 | 在品类搜索框输入关键词 | 实时过滤品类列表 |
| TS-001-04 | 选择一个品类 | 品类字段显示选中值 |
| TS-001-05 | 选择品类后切换 | 提示"切换品类将清空已填属性" |
| TS-001-06 | 品类列表为空 | 显示"暂无品类，请联系管理员" |

#### 验收条件
- [ ] 品类下拉框加载时间≤1 秒
- [ ] 支持键盘上下键选择
- [ ] 选中品类后自动触发属性模板加载
- [ ] 已禁用的品类不显示在列表中

### AC-002: 属性模板自动加载

**对应故事：** US-001, US-003

#### 测试场景

| 场景编号 | 场景描述 | 预期结果 |
|----------|----------|----------|
| TS-002-01 | 选择美妆品类 | 显示美妆类属性字段（规格、保质期、适用肤质等） |
| TS-002-02 | 选择电器品类 | 显示电器类属性字段（功率、电池容量、防水等级等） |
| TS-002-03 | 切换不同品类 | 属性表单动态更新 |
| TS-002-04 | 属性模板有必填字段 | 必填字段显示红色星号标记 |
| TS-002-05 | 属性模板有默认值 | 字段自动填充默认值 |
| TS-002-06 | 网络异常加载失败 | 显示"属性模板加载失败，请重试" |

#### 验收条件
- [ ] 属性表单渲染时间≤1 秒
- [ ] 字段类型正确渲染（文本框、数字框、下拉框等）
- [ ] 必填字段有明确标识
- [ ] 加载失败有错误提示和重试机制

## 验收测试流程

### 阶段一：功能验收（P0）

1. **环境准备**
   - 部署测试环境
   - 准备测试数据（品类、模板）
   - 配置测试账号（采购专员、采购经理、管理员）

2. **执行测试**
   - 按 AC-001 到 AC-015 执行测试
   - 记录测试结果和缺陷
   - 缺陷修复后回归测试

3. **验收标准**
   - P0 验收标准通过率 100%
   - 严重缺陷 0 个
   - 一般缺陷≤5 个

### 阶段二：集成验收（P1）

1. **集成测试**
   - 与备货计划模块集成
   - 与用户认证系统集成
   - 与文件存储服务集成

2. **执行测试**
   - 按 AC-013、AC-016、AC-018 执行测试
   - 端到端业务流程测试

3. **验收标准**
   - P1 验收标准通过率≥95%
   - 集成问题全部解决

### 阶段三：用户验收（UAT）

1. **用户测试**
   - 邀请真实用户参与测试
   - 收集用户反馈
   - 优化用户体验

2. **验收标准**
   - 用户满意度≥85%
   - 核心业务流程顺畅

## 缺陷等级定义

| 等级 | 定义 | 处理要求 |
|------|------|----------|
| 严重 | 核心功能不可用，数据丢失 | 必须修复，否则不能上线 |
