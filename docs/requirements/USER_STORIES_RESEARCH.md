## 用户故事详情

### US-001: 创建产品调研

**作为** 采购经理  
**我希望** 创建新产品调研并选择品类  
**以便** 开始产品选型工作

#### 验收标准
- [ ] 可选择产品品类（美妆/个护/电器等）
- [ ] 自动加载该品类的属性模板
- [ ] 可录入产品基本信息（名称、品牌、平台、价格等）
- [ ] 可上传产品图片
- [ ] 保存后状态为草稿

#### 备注
- 品类选择使用下拉菜单，支持搜索
- 创建成功后跳转到编辑页面

---

### US-002: 录入产品基本信息

**作为** 采购专员  
**我希望** 录入产品的基本信息  
**以便** 记录产品的核心商业数据

#### 验收标准
- [ ] 必填字段：产品名称、品牌、平台、采购价、售价
- [ ] 选填字段：产品链接、供应商信息、备注
- [ ] 价格字段支持小数，最多 2 位
- [ ] 平台字段为下拉选择（Amazon、TikTok、独立站等）
- [ ] 产品链接自动验证 URL 格式
- [ ] 必填字段未填写时禁止保存

#### 备注
- 采购价和售价应显示货币符号（默认 USD）
- 产品链接点击可在新窗口打开

---

### US-003: 录入动态属性

**作为** 采购专员  
**我希望** 根据品类自动显示对应的属性字段  
**以便** 录入该品类的专业参数

#### 验收标准
- [ ] 选择品类后自动渲染对应的属性表单
- [ ] 支持多种字段类型：文本、数字、日期、下拉选择、多选、布尔值
- [ ] 必填属性用红色星号标记
- [ ] 下拉选择和多选字段显示预设选项
- [ ] 未填写必填属性时禁止提交
- [ ] 属性值可修改和重新保存

#### 备注
- 不同品类的属性模板由管理员预先配置
- 属性模板版本变更不影响历史数据

---

### US-004: 上传产品图片

**作为** 采购专员  
**我希望** 上传产品图片  
**以便** 直观展示产品外观和细节

#### 验收标准
- [ ] 支持上传图片格式：JPG、PNG、WEBP
- [ ] 单张图片最大 5MB
- [ ] 最多上传 9 张图片
- [ ] 上传后显示缩略图预览
- [ ] 可删除已上传的图片
- [ ] 可调整图片顺序（拖拽或上下移动）
- [ ] 上传进度条显示

#### 备注
- 第一张图片作为封面图
- 图片存储到对象存储服务

### US-005: 保存和管理草稿

**作为** 采购专员  
**我希望** 随时保存草稿并稍后继续编辑  
**以便** 分多次完成产品调研录入

#### 验收标准
- [ ] 支持手动保存草稿
- [ ] 编辑过程中自动保存（每 60 秒）
- [ ] 草稿列表显示所有未完成的调研
- [ ] 草稿可继续编辑
- [ ] 草稿可删除
- [ ] 草稿状态明确标识

#### 备注
- 自动保存时显示"已保存"提示
- 草稿超过 90 天未编辑发送提醒

### US-006: 创建对比列表

**作为** 采购经理  
**我希望** 选择多个产品创建对比列表  
**以便** 横向评估候选产品

#### 验收标准
- [ ] 可从调研列表中选择 2-10 个产品
- [ ] 可为对比列表命名（如"美容仪对比-Q1"）
- [ ] 支持保存对比列表
- [ ] 支持复制已有对比列表
- [ ] 对比列表可删除
- [ ] 对比列表仅创建人可见（或团队共享，可配置）

#### 备注
- 选择产品时显示产品名称、品牌、价格等关键信息
- 支持按品类筛选可对比产品

---

### US-007: 查看产品对比

**作为** 采购经理  
**我希望** 以表格形式查看多个产品的对比  
**以便** 快速识别产品差异和优劣

#### 验收标准
- [ ] 表格形式展示：每列一个产品，每行一个属性
- [ ] 自动对齐相同属性进行对比
- [ ] 关键差异高亮显示（价格差异>20%、评分差异>1 星等）
- [ ] 支持自定义显示/隐藏对比字段
- [ ] 支持按任意字段排序
- [ ] 支持导出对比结果为图片或 PDF

#### 备注
- 默认显示基本信息和关键属性
- 高亮规则可配置

---

### US-008: 提交调研结论

**作为** 采购专员  
**我希望** 提交产品调研的最终结论  
**以便** 为备货决策提供依据

#### 验收标准
- [ ] 结论类型：推荐（优先备货）、备选（可考虑）、淘汰（不推荐）
- [ ] 必须填写结论理由说明
- [ ] 支持多维度打分（1-5 分）：市场需求、利润空间、竞争程度、供应链稳定性
- [ ] 自动计算综合得分（权重可配置）
- [ ] 提交后状态变更为"待审核"
- [ ] 提交后不可直接修改，需走变更流程

#### 备注
- 综合得分计算公式：∑(维度得分 × 权重)
- 默认权重：市场需求 30%、利润空间 30%、竞争程度 20%、供应链稳定性 20%

---

### US-009: 审核调研结论

**作为** 采购经理  
**我希望** 审核采购专员提交的调研结论  
**以便** 确保决策质量

#### 验收标准
- [ ] 可查看待审核的调研列表
- [ ] 可查看调研详情和结论理由
- [ ] 审核意见：通过、驳回（需填写原因）、修改后通过
- [ ] 审核通过后状态变更为"已完成"
- [ ] 驳回后状态变更为"调研中"，返回给原提交人
- [ ] 审核记录可追溯（审核人、时间、意见）

#### 备注
- 驳回时需填写具体原因和改进建议
- 支持批量审核

### US-010: 关联备货计划

**作为** 采购经理  
**我希望** 将推荐产品关联到备货计划  
**以便** 启动采购流程

#### 验收标准
- [ ] 仅"推荐"结论的产品可关联备货计划
- [ ] 关联时选择或创建备货计划
- [ ] 一个产品可关联多个备货计划（不同批次）
- [ ] 关联时自动同步产品基本信息和属性
- [ ] 备货计划可查看关联的产品调研详情
- [ ] 产品调研页面显示关联的备货计划列表

#### 备注
- 备货计划模块需同步开发
- 关联后产品信息变更需评估对备货计划的影响

### US-011: 管理品类模板

**作为** 系统管理员  
**我希望** 创建和管理产品品类  
**以便** 规范产品分类体系

#### 验收标准
- [ ] 可创建新品类（名称、编码、描述）
- [ ] 可编辑品类信息
- [ ] 可禁用品类（不影响历史数据）
- [ ] 品类支持层级结构（一级 → 二级）
- [ ] 品类列表可搜索和筛选
- [ ] 删除品类前检查是否有关联调研

#### 备注
- 品类编码用于系统集成和报表
- 禁用而非删除，保证数据完整性

### US-012: 管理属性模板

**作为** 系统管理员  
**我希望** 为品类配置属性模板  
**以便** 定义该品类的标准评估维度

#### 验收标准
- [ ] 可为品类添加、编辑、删除属性字段
- [ ] 支持字段类型：文本、数字、日期、下拉选择、多选、布尔值
- [ ] 可设置字段是否必填
- [ ] 可设置默认值
- [ ] 下拉选择和多选可配置选项列表
- [ ] 属性模板版本管理（修改后生成新版本）
- [ ] 历史调研数据保持原模板版本

#### 备注
- 版本变更时提示影响范围
- 建议变更在业务低峰期进行

### US-013: 搜索和筛选调研

**作为** 采购专员  
**我希望** 搜索和筛选产品调研  
**以便** 快速找到目标调研

#### 验收标准
- [ ] 支持按产品名称/品牌搜索
- [ ] 支持按品类筛选
- [ ] 支持按状态筛选（草稿/调研中/待审核/已完成）
- [ ] 支持按结论筛选（推荐/备选/淘汰）
- [ ] 支持按创建时间范围筛选
- [ ] 支持按创建人筛选
- [ ] 搜索结果可排序（时间、名称、价格等）

#### 备注
- 支持组合筛选条件
- 筛选条件可保存为常用视图

### US-014: 导出调研数据

**作为** 采购经理  
**我希望** 导出产品调研数据  
**以便** 进行离线分析和汇报

#### 验收标准
- [ ] 支持导出单个调研详情（PDF 格式）
- [ ] 支持导出调研列表（Excel 格式）
- [ ] 支持导出对比结果（图片/PDF）
- [ ] 导出内容包含基本信息、属性、图片、结论
- [ ] 导出文件命名规范（含日期和产品名）
- [ ] 大批量导出时后台处理，完成后通知

#### 备注

- Excel 导出支持自定义字段

### US-015: 查看操作历史

**作为** 采购经理  
**我希望** 查看产品调研的操作历史  
**以便** 追溯数据变更和审计

#### 验收标准
- [ ] 记录关键操作：创建、修改、删除、状态变更、结论提交、审核
- [ ] 操作历史包含：操作人、操作时间、操作类型、变更内容
- [ ] 操作历史按时间倒序排列
- [ ] 支持查看操作详情（修改前后的值）
- [ ] 操作历史不可删除和修改

#### 备注
- 操作历史保留至少 2 年
- 敏感操作（删除）需额外审批
