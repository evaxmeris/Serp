## 🔍 问题排查

### 如果遇到问题

1. **页面显示"加载中..."**
   - 原因：浏览器缓存
   - 解决：强制刷新（Cmd+Shift+R）

2. **对话框不弹出**
   - 原因：JavaScript 错误
   - 解决：打开控制台（F12）查看错误

3. **API 请求失败**
   - 原因：服务器未启动
   - 解决：检查开发服务器状态
