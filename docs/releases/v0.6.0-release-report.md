## ✨ 新增功能

### 1. 财务报表模块

#### 利润报表
- **API:** `/api/v1/reports/profit`
- **页面:** `/reports/profit`
- **功能:** 收入、成本、毛利润、净利润分析

#### 销售报表
- **API:** `/api/v1/reports/sales`
- **页面:** `/reports/sales`
- **功能:** 销售趋势、客户分析、产品分析

#### 库存报表
- **API:** `/api/v1/reports/inventory`
- **页面:** `/reports/inventory`
- **功能:** 库存周转、库龄分析、呆滞库存

#### 采购报表
- **API:** `/api/v1/reports/purchase`
- **页面:** `/reports/purchase`
- **功能:** 采购分析、供应商绩效

#### 现金流报表
- **API:** `/api/v1/reports/cashflow`
- **页面:** `/reports/cashflow`
- **功能:** 现金流入流出、应收应付

#### 自定义报表
- **API:** `/api/v1/reports/custom`
- **页面:** `/reports/custom`
- **功能:** 自定义报表配置和管理

### 2. 报表辅助功能

#### 报表导出
- **API:** `/api/v1/reports/export`
- **功能:** 导出 Excel/CSV 格式报表

#### 报表订阅
- **API:** `/api/v1/reports/subscribe`
- **功能:** 定时推送报表到邮箱

#### 报表调度
- **API:** `/api/v1/reports/schedule`
- **功能:** 配置报表生成计划

#### 报表仪表盘
- **API:** `/api/v1/reports/dashboard`
- **页面:** `/reports/dashboard`
- **功能:** 财务报表总览

## 📝 重要教训

### Sprint 6 开发中的 Schema 破坏错误

**事件：** 开发中错误删除了 Sprint 5 的核心字段

**影响：**
- 10+ 个 API 被破坏
- Docker 构建失败
- 需要回滚重新开发

**改进措施：**
1. ✅ Schema 修改前必须阅读现有代码
2. ✅ 禁止删除现有字段（只能新增）
3. ✅ 必须进行影响分析和回归测试
4. ✅ 错误教训已写入 AGENTS.md
