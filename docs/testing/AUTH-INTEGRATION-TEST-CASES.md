### TC-AUTH-001: 用户登录成功

**测试目的**: 验证正确用户名密码可以成功登录

**前置条件**:
- 测试用户 `test@example.com` 已存在
- 状态: `ACTIVE`
- 密码 `Password123!` 正确

**测试步骤**:
1. 发送 POST 请求到 `/api/auth/login`
   ```json
   {
     "email": "test@example.com",
     "password": "Password123!"
   }
   ```
2. 检查响应状态码
3. 检查响应体
4. 检查响应头 Set-Cookie

**预期结果**:
- ✅ 响应状态码: `200 OK`
- ✅ 响应 JSON:
  ```json
  {
    "success": true,
    "user": {
      "id": "...",
      "email": "test@example.com",
      "name": "Test User",
      "role": "USER"
    }
  }
  ```
- ✅ Set-Cookie 包含 `auth-token`
- ✅ Cookie 属性包含 `HttpOnly`
- ✅ Cookie 属性包含 `SameSite=lax`
- ✅ Cookie `max-age=604800` (7 天)
- ✅ 开发环境 `Secure` 属性不设置，生产环境自动设置

**测试类型**: 自动化集成测试  
**优先级**: 🔴 P0  
**实际结果**: ⏳ 待执行
