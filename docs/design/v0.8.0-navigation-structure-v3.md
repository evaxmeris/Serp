# Trade ERP v0.8.0 - 导航系统设计文档

## 1. 系统概述

### 1.1 设计目标
- 支持 10 种角色权限控制
- 可折叠/展开的左侧导航栏
- 清晰的信息架构
- 响应式多端适配
- 图标化视觉引导

### 1.2 10 种系统角色定义

| 角色ID | 角色名称 | 职责描述 |
|--------|----------|----------|
| 1 | 超级管理员 | 系统全权限，配置管理 |
| 2 | 公司老板 | 查看全公司数据，报表分析 |
| 3 | 运营经理 | 客户、销售、采购、库存管理 |
| 4 | 销售业务员 | 客户跟进、订单处理 |
| 5 | 采购专员 | 供应商管理、采购下单 |
| 6 | 仓库管理员 | 库存管理、出入库操作 |
| 7 | 财务会计 | 财务管理、报表核算 |
| 8 | 绩效管理员 | 绩效设置、提成计算 |
| 9 | 平台运营专员 | 多平台账号管理 |
| 10 | 产品开发专员 | 产品信息维护、开发跟进 |

## 2. 左侧导航栏设计

### 2.1 基础结构

```
┌─────────────────┐
│  Logo      ≡   │ ← 顶部Logo + 折叠按钮
├─────────────────┤
│                 │
│  🧑‍💼 客户管理    │ ← 一级菜单（带图标）
│   ├─ 客户列表   │ ← 二级菜单
│   └─ 客户跟进   │
│                 │
│  💼 销售管理    │
│   ├─ 订单列表   │
│   ├─ 订单创建   │
│   └─ 发货管理   │
│                 │
│  ...            │
├─────────────────┤
│ 👤 当前用户 ▼   │ ← 底部用户信息
└─────────────────┘
```

### 2.2 交互行为

- **折叠状态**：只显示图标，鼠标悬浮显示文字提示
- **展开状态**：显示图标 + 文字
- **自动折叠**：平板端默认折叠，移动端自动隐藏（需侧滑呼出）
- **记住状态**：用户偏好存储在 localStorage

## 5. 响应式导航设计

### 5.1 断点设计

| 屏幕宽度 | 导航模式 | 默认状态 |
|----------|----------|----------|
| ≥ 1200px (桌面) | 左侧固定 + 展开 | 展开 |
| 768px - 1199px (平板) | 左侧可折叠 | 折叠 |
| < 768px (手机) | 悬浮抽屉 + 遮罩 | 关闭，需侧滑/点击呼出 |

### 5.2 交互适配

**桌面端（≥1200px）**
- 导航宽度：展开 240px / 折叠 64px
- 内容区域自适应留白
- 支持手动切换展开/折叠

**平板端（768px-1199px）**
- 导航宽度：展开 240px / 折叠 64px
- 默认折叠，节省内容空间
- 用户可手动展开

**移动端（<768px）**
- 导航从左侧滑入，占屏幕 80% 宽度
- 背景半透明遮罩，点击遮罩关闭
- 顶部有关闭按钮

## 7. 权限控制逻辑

### 7.1 前端控制流程

```javascript
1. 用户登录成功 → 获取角色ID
2. 根据角色ID → 获取该角色可见菜单树
3. 渲染导航 → 只渲染有权限的菜单项
4. 路由守卫 → 检查当前路由是否在权限列表中
5. 无权限 → 跳转403或首页
```

### 7.2 后端双重验证

- 前端隐藏 + 后端路由权限验证
- 即使手动输入URL，后端也会验证权限
- 数据级别权限：业务员只能看到自己的客户/订单
