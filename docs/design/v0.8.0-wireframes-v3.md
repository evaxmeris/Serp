## 2. 响应式布局设计

### 2.1 断点定义

| 断点 | 屏幕宽度 | 布局模式 | 导航模式 |
|------|----------|----------|----------|
| 桌面端 | ≥ 1200px | 三栏？不，两栏：侧边导航 + 内容 | 左侧固定展开 |
| 平板端 | 768px - 1199px | 两栏：折叠侧边 + 内容 | 左侧折叠（仅图标） |
| 移动端 | < 768px | 单栏：内容全屏 | 隐藏，侧滑呼出 |

### 2.2 网格系统

采用 **Tailwind CSS 断点 + 12 列网格**：

```css
.container {
  width: 100%;
  margin: 0 auto;
  padding: 16px;
}

/* 12列网格 */
.row {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 16px;
}

/* 响应式列宽 */
@media (min-width: 1200px) {
  .card-4 { grid-column: span 3; } /* 4列 */
  .chart-2 { grid-column: span 6; } /* 两图各6 */
  .quick-actions { grid-column: span 12; }
  .list-2 { grid-column: span 6; }
}

@media (min-width: 768px) and (max-width: 1199px) {
  .card-4 { grid-column: span 6; } /* 2行 × 2列 */
  .chart-2 { grid-column: span 12; } /* 两个图上下排列 */
  .quick-actions { grid-column: span 12; }
  .list-2 { grid-column: span 12; } /* 列表上下排列 */
}

@media (max-width: 767px) {
  .card-4 { grid-column: span 12; } /* 1列纵向排列 */
  .chart-2 { grid-column: span 12; }
  .quick-actions { grid-column: span 12; }
  .list-2 { grid-column: span 12; }
  /* 快捷操作：2行 × 3列 → 3行 × 2列 */
}
```

## 4. 布局规范

### 4.1 间距系统

采用 **8px 网格**：

| 元素 | 间距 |
|------|------|
| 页面边距 | 16px |
| 区块间距 | 24px |
| 卡片间距 | 16px |
| 行高（文字） | 1.5 |
| 表单标签与输入框间距 | 8px |
| 表单项间距 | 16px |
| 按钮内边距（上下） | 8px |
| 按钮内边距（左右） | 16px |

### 4.2 字体大小系统

| 用途 | px | rem |
|------|----|-----|
| H1 标题 | 28px | 1.75rem |
| H2 标题 | 24px | 1.5rem |
| H3 标题 | 20px | 1.25rem |
| 正文 | 16px | 1rem |
| 次级文字 | 14px | 0.875rem |
| 小字/辅助 | 12px | 0.75rem |

### 4.3 边框圆角

- 按钮：6px
- 卡片：8px
- 输入框：6px
- 模态框：12px

## 6. 组件规范

### 6.1 按钮规范

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| 主按钮 | 蓝色背景白色文字 | 主要操作（新建、提交、确认） |
| 次按钮 | 灰色边框灰色文字 | 次要操作（取消、返回） |
| 危险按钮 | 红色背景白色文字 | 删除、禁用等危险操作 |
| 文字按钮 | 只有文字蓝色 | 链接类操作 |

**尺寸规范：**
- 大按钮：padding 12px 24px，字体 16px
- 中按钮：padding 8px 16px，字体 14px （默认）
- 小按钮：padding 4px 12px，字体 12px

### 6.2 表单规范

- 标签：必填项加红色星号 `*`
- 输入框：下方留出错提示空间
- 分组：同组表单用卡片分隔
- 操作栏：底部固定（长表单），保存/取消按钮

### 6.3 表格规范

- 表头固定
- 行高：48px
- 隔行变色（可选）
- 横向滚动（列多的情况）
- 移动端：卡片式展示每行

### 6.4 模态框规范

- 宽度：桌面端 500px-800px（根据内容）
- 移动端：90% 宽度，距离顶部 16px
- 顶部标题栏 + 关闭按钮
- 底部操作栏（确定/取消）
- 点击遮罩关闭（可选）
