# Trade ERP v0.8.0 - 导航系统设计（更新版）

## 设计概述

### 导航架构
- **左侧固定导航栏** + **顶部面包屑导航** + **用户信息栏**
- 支持**折叠/展开**动画切换
- 支持**角色权限动态菜单显示**
- 响应式设计适配多终端

### 9 种系统角色

| 角色ID | 角色名称 | 职责描述 |
|--------|----------|----------|
| 1 | 超级管理员 | 全系统访问权限 |
| 2 | 公司老板 | 查看报表、审批权限 |
| 3 | 运营经理 | 销售、采购、库存管理 |
| 4 | 销售运营 | 客户管理、订单处理 |
| 5 | 采购专员 | 供应商管理、采购下单 |
| 6 | 仓库管理员 | 库存管理、出入库 |
| 7 | 财务会计 | 财务管理、报销核算 |
| 8 | 产品开发 | 产品管理、SKU维护 |
| 9 | 平台运营 | 多平台账号管理 |

## 交互设计

### 折叠/展开动画
- **默认状态**：桌面端展开，平板端折叠，移动端隐藏
- **动画时长**：250ms 缓动效果
- **触发方式**：点击汉堡按钮 ⟲ / 拖拽边缘
- **记忆功能**：记住用户上次状态，LocalStorage 持久化

### 菜单项交互
- **Hover 效果**：背景浅蓝高亮，左侧边栏蓝色指示条
- **Active 状态**：当前页面蓝色背景 + 白色文字
- **二级菜单**：点击主项展开/折叠，带有旋转箭头动画
- **Badge 提醒**：待办数量红色角标显示（如审批待办、库存预警）

### 面包屑导航
- **位置**：顶部内容区上方
- **显示层级**：首页 > 一级菜单 > 二级菜单 > 当前页面
- **可点击**：所有层级均可点击跳转
- **响应式**：移动端省略中间层级，只显示首页 > 当前页

## 响应式导航设计

### Breakpoints
| 断点 | 导航行为 |
|------|----------|
| ≥1200px (桌面) | 左侧展开，可折叠 |
| 768px - 1199px (平板) | 左侧默认折叠，点击展开浮层 |
| <768px (手机) | 隐藏，点击汉堡按钮从左侧滑出（遮罩层） |

### 移动端导航
- 宽度：85% 屏幕宽度
- 滑入动画：从左向右
- 遮罩层：半透明黑色背景，点击遮罩关闭
- 关闭按钮：右上角 X 按钮

## 菜单搜索功能

- **触发**：顶部搜索框或快捷键 `/`
- **功能**：搜索所有可见菜单项（按当前角色过滤）
- **快捷键**：上下箭头选择，回车跳转
- **结果**：显示匹配的菜单路径，高亮关键词
