---

## 🎉 新功能

### 1. 批量操作功能

#### 产品管理
- **批量导入** - 支持 Excel/CSV 格式，单次最多 1000 条
- **批量导出** - CSV 格式，支持筛选
- **批量删除** - 关联数据检查，防止误删

#### 订单管理
- **批量确认** - 一键确认多个订单
- **批量发货** - 批量创建出库单，支持物流单号

#### 客户管理
- **批量导入** - 邮箱重复检测
- **批量导出** - CSV 格式
- **批量打标签** - 添加/移除/替换三种模式

### 2. UI/UX 优化

- **AuthGuard 组件** - 未登录时显示友好提示
- **ErrorBoundary 组件** - 捕获错误并显示重试按钮
- **角色权限检查** - 无权限时显示清晰提示

### API 设计

{
  "ids": ["id1", "id2"],
  "mode": "create|update|delete"
}
